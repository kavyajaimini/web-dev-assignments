Introduction to Semantic HTML

Semantic HTML gives meaning to our web page content,helps browsers and search engines interpret the structure of each section.

Benefits are:-
  • Improved Accessibility to the users.
  • SEO friendly.
  • Clean and Maintainable Code.

Conclusion:

Using semantic HTML elements like <article>, <nav> etc. will improve the user experience.
